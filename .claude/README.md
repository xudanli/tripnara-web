# .claude 目录结构说明

本目录用于存储 Claude Agent 协作相关的文档和资料。

## 文件夹用途

### `agents/` 文件夹
- **用途**：存储 Agent 角色定义文档
- **内容**：各 Agent 的角色定位、职责、工作流程等定义文档
- **说明**：这些文档用于指导 Claude 在代码编写和项目决策中扮演不同角色，与系统实现的智能体（Agent UI）**没有关系**
- **示例**：
  - `Agent 编排者.md` - Agent 编排者的角色定义
  - `产品经理.md` - 产品经理 Agent 的角色定义
  - `资深前端工程师.md` - 资深前端工程师 Agent 的角色定义

### `改动资料/` 文件夹
- **用途**：存储每个角色总结之后的文件
- **内容**：各角色完成工作后生成的总结文档、决策记录等
- **说明**：用于记录项目过程中各角色的工作成果和总结
- **示例**：
  - `TRIPNARA-PRODUCT-STORY.md` - 产品故事总结
  - `TRIPNARA-MOBILE-PRODUCT-VISION.md` - 移动端产品愿景总结

## 注意事项

- `agents/` 文件夹中的文档是**角色定义文档**，用于指导 Claude 在协作中的行为
- `agents/` 文件夹中的内容**不是系统实现的智能体代码**，而是 Claude 协作的工作指导
- `改动资料/` 文件夹用于存储**工作成果和总结**，记录项目过程中的重要决策和变更
