# Planning Assistant V2 - 视觉设计审核报告

**审核日期**: 2026-02-08  
**审核人**: 视觉设计师  
**审核范围**: MCP 服务数据展示组件

---

## 📋 审核概览

### ✅ 优点
1. **组件化设计良好** - 各功能模块独立，易于维护
2. **使用统一的设计系统** - 使用 shadcn/ui 组件库，保持一致性
3. **响应式布局** - 支持移动端和桌面端
4. **交互反馈** - 有 hover 效果和过渡动画

### ⚠️ 需要改进的问题

#### 1. **间距不一致** 🔴 高优先级
- **问题**: 不同组件的间距不统一
  - `HotelList`: `space-y-3` (12px)
  - `RestaurantList`: `space-y-3` (12px)
  - `SearchResults`: `space-y-2` (8px)
  - `MCPDataDisplay` 中的航班/铁路: `space-y-2` (8px)
- **建议**: 统一使用 `space-y-3` (12px) 作为卡片间距

#### 2. **卡片内边距不一致** 🔴 高优先级
- **问题**: CardContent 的 padding 不统一
  - 推荐卡片: 默认 `p-6 pt-0`
  - 酒店/餐厅: `space-y-2` + 默认 padding
  - 航班/铁路: `pt-4` (16px)
  - 天气: `pt-6` (24px)
- **建议**: 统一使用 CardContent 的默认 padding (`p-6 pt-0`)，或明确规范

#### 3. **图标大小不一致** 🟡 中优先级
- **问题**: 
  - 推荐卡片: `w-4 h-4` (16px)
  - 酒店/餐厅: `w-4 h-4` (16px)
  - 航班/铁路: `w-5 h-5` (20px)
  - 天气: `w-8 h-8` (32px)
- **建议**: 
  - 卡片标题图标: `w-4 h-4` (16px)
  - 功能图标（天气、翻译等）: `w-5 h-5` (20px)
  - 大型展示图标: `w-8 h-8` (32px)

#### 4. **价格信息不够突出** 🟡 中优先级
- **问题**: 价格信息在卡片中不够显眼
- **建议**: 
  - 使用更大的字体 (`text-base` 或 `text-lg`)
  - 使用主色调 (`text-primary`)
  - 添加货币符号图标

#### 5. **评分显示不一致** 🟡 中优先级
- **问题**: 
  - 酒店/餐厅: 使用 Badge + Star 图标
  - 推荐卡片: 使用 Badge + Star 图标
  - 但样式略有不同
- **建议**: 统一评分显示组件

#### 6. **图片网格需要优化** 🟡 中优先级
- **问题**: 图片网格在移动端可能显示不佳
- **建议**: 
  - 移动端: `grid-cols-2`
  - 平板: `md:grid-cols-3`
  - 桌面: `lg:grid-cols-4`
  - 添加图片加载占位符

#### 7. **缺少空状态处理** 🟢 低优先级
- **问题**: 某些组件没有空状态提示
- **建议**: 添加友好的空状态提示

#### 8. **可访问性** 🟡 中优先级
- **问题**: 
  - 缺少 aria-label
  - 链接缺少描述性文本
- **建议**: 添加适当的 aria-label 和描述

---

## 🎨 设计规范建议

### 间距系统
```
- 卡片间距: 12px (space-y-3)
- 卡片内间距: 24px (p-6)
- 元素间距: 8px (gap-2) 或 12px (gap-3)
- 标题与内容间距: 12px (mt-3)
```

### 字体大小
```
- 卡片标题: text-base (16px) 或 text-lg (18px)
- 正文: text-sm (14px)
- 辅助信息: text-xs (12px)
- 价格: text-base (16px) 或 text-lg (18px)
```

### 颜色使用
```
- 主色调: text-primary (用于链接、重要信息)
- 次要信息: text-muted-foreground
- 评分: text-yellow-500 (Star 图标)
- 价格: text-primary (突出显示)
```

### 图标规范
```
- 卡片标题图标: w-4 h-4 (16px)
- 功能图标: w-5 h-5 (20px)
- 大型展示图标: w-8 h-8 (32px)
- 小型辅助图标: w-3 h-3 (12px)
```
