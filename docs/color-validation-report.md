## ⚠️ 边框颜色说明

### 现状
文档中**未指定**边框颜色，但代码中使用了边框颜色。

### 当前实现
```typescript
'very-low': {
  borderColor: 'border-[#C8E6C9]',    // 基于 #4CAF50 的浅色变体
},
'low-medium': {
  borderColor: 'border-[#DCEDC8]',   // 基于 #9CCC65 的浅色变体
},
'medium': {
  borderColor: 'border-[#FFF59D]',    // 基于 #FFC107 的浅色变体
},
'medium-high': {
  borderColor: 'border-[#FFE0B2]',   // ⚠️ 注意：这是高风险等级的背景色
},
'high': {
  borderColor: 'border-[#FFCC80]',   // 基于 #FF6F00 的浅色变体
},
'very-high': {
  borderColor: 'border-[#FFCDD2]',   // 基于 #F44336 的浅色变体
},
```

### 发现的问题

**问题：颜色值重复使用**
- `medium-high` 的 `borderColor` 是 `#FFE0B2`
- `high` 的 `bgColor` 也是 `#FFE0B2`

**评估：**
- ⚠️ 这可能导致视觉混淆（中高风险的边框和高风险的背景是同一颜色）
- 建议：为 `medium-high` 使用不同的边框颜色

### 建议修复

**选项1：使用文档中高风险等级的背景色作为中高风险的边框**
```typescript
'medium-high': {
  borderColor: 'border-[#FFE0B2]',  // 当前：使用高风险背景色
},
```
**问题：** 与高风险背景色重复

**选项2：基于中高风险主色推导边框颜色**
```typescript
'medium-high': {
  borderColor: 'border-[#FFB74D]',  // 基于 #FF9800 的浅色变体
},
```
**优点：** 与主色系统一致，避免重复

**选项3：使用中高风险背景色的深色变体**
```typescript
'medium-high': {
  borderColor: 'border-[#FFCC80]',  // 基于 #FFF3E0 的深色变体
},
```
**优点：** 与背景色系统一致
