## 📁 文件变更

### 修改的文件

**`src/pages/trips/[id].tsx`** - 行程详情页

#### 1. 添加预算标签页

**位置**: 标签页列表（TabsList）

**变更内容**:
- 在标签页列表中添加"预算"标签
- 使用 Wallet 图标
- 添加对应的 TabsContent，直接渲染预算页面组件

**代码变更**:
```tsx
// 添加图标导入
import { Wallet, ExternalLink } from 'lucide-react';
import TripBudgetPage from './budget';

// 添加标签页
<TabsTrigger value="budget">
  <Wallet className="w-4 h-4 mr-2" />
  预算
</TabsTrigger>

// 添加标签页内容
<TabsContent value="budget" className="mt-0 p-0">
  {id ? (
    <TripBudgetPage />
  ) : (
    <div className="p-6 text-center text-muted-foreground">
      <p>无法加载预算信息</p>
    </div>
  )}
</TabsContent>
```

#### 2. 添加行程健康度快速入口

**位置**: 行程健康度卡片（Health Card）的预算维度

**变更内容**:
- 将预算维度标签改为可点击按钮
- 点击后切换到预算标签页
- 添加 ExternalLink 图标提示

**代码变更**:
```tsx
{key === 'budget' && (
  <button
    onClick={() => {
      setActiveTab('budget');
    }}
    className="flex items-center gap-1 hover:text-primary transition-colors"
  >
    预算
    <ExternalLink className="w-3 h-3" />
  </button>
)}
```

## 📍 访问路径

### 方式 1: 标签页导航

1. 打开行程详情页：`/trips/:id`
2. 点击顶部标签页中的"预算"标签
3. 直接进入预算管理页面

### 方式 2: 快速入口

1. 打开行程详情页：`/trips/:id`
2. 在"总览"标签页的右侧栏找到"行程健康度"卡片
3. 点击"预算"维度（带链接图标）
4. 自动切换到预算标签页
