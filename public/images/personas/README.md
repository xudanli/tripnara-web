# Persona Images 形象图片

此目录用于存放三个形象（Persona）的PNG图片文件。

## 文件命名

请将图片文件命名为：

- `abu.png` - Abu（北极熊）的形象图片
- `dr-dre.png` - Dr.Dre（牧羊犬）的形象图片  
- `neptune.png` - Neptune（海獭）的形象图片

## 使用方式

在代码中可以通过以下路径访问：

```tsx
// 在组件中使用
<img src="/images/personas/abu.png" alt="Abu" />
<img src="/images/personas/dr-dre.png" alt="Dr.Dre" />
<img src="/images/personas/neptune.png" alt="Neptune" />
```

## 图片要求

- 格式：PNG（支持透明背景）
- 建议尺寸：200x200px 或更大（保持宽高比）
- 风格：符合品牌黑白线稿风格，或彩色卡通风格（根据设计稿）
